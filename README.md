# ₿ 比特币定投计划 - DCA 投资计算器

一个现代化的比特币定期定额投资（DCA）计划管理网站。帮助用户轻松跟踪和分析长期比特币投资。

## 🌟 功能特性

### 📊 定投计算器
- 灵活的投资金额输入
- 实时 BTC 价格同步
- 自动计算获得的 BTC 数量
- 快捷按钮（$500、$1000）
- 日期选择功能

### 💰 投资组合面板
- 总投入金额统计
- 持有 BTC 数量显示
- 当前组合价值计算
- 绝对收益和亏损展示
- 收益率百分比

### 📈 投资分析
- 平均成本计算
- 当前价格显示
- 成本对比分析
- 价格趋势图表
- 月度投资统计

### 📝 投资记录管理
- 完整的投资历史
- 每笔投资的详细信息
- 单笔收益计算
- 灵活的删除功能
- 数据统计汇总

### 📱 响应式设计
- 桌面端优化
- 平板端适配
- 移动端友好
- 深色主题设计

### 💾 数据持久化
- 浏览器本地存储
- 无需后端服务
- 数据隐私保护
- 自动保存

## 🚀 快速开始

### 环境要求
- Node.js 18+

1. **克隆或下载项目**
   ```bash

   ```

2. **安装依赖**
   ```bash

   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **打开浏览器**

## 🎯 使用指南

### 添加投资记录

1. 切换到"定投计算器"标签页
2. 选择投资日期
3. 输入投资金额（USD）
4. 输入 BTC 买入价格或点击"用当前价格"
5. 查看获得的 BTC 数量
6. 点击"添加记录"完成

### 查看投资分析

1. 切换到"投资分析"标签页
2. 查看平均成本与当前价格对比
3. 查看价格趋势图表
4. 查看月度投资统计

### 管理投资记录

1. 切换到"投资记录"标签页
2. 查看所有投资的详细信息
3. 实时查看每笔投资的收益情况
4. 支持删除不需要的记录

## 📈 特色功能详解

### DCA 定投的优势

1. **分散风险**：定期买入，平滑成本曲线
2. **避免择时**：无需预测市场顶部和底部
3. **心理建设**：规律投资培养纪律性
4. **长期收益**：适合 5+ 年长期持有

### 平均成本计算

平均成本 = 总投入金额 / 累计 BTC 数量

这反映了您实际的平均买入价格。

### 收益计算

- **绝对收益** = 当前价值 - 总投入
- **收益率** = 绝对收益 / 总投入 × 100%

## 🌐 实时数据源

- 实时更新
